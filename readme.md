# 🤖 Cyber Nexus - Cyborg Themed Landing Page

## 📖 Project Overview

Cyber Nexus is a futuristic, responsive landing page developed for a Tech Fest based on the **Cyborg Theme**. The website showcases modern UI/UX design with cyberpunk-inspired colors, glowing effects, animations, and responsive layouts.

---

## ✨ Features

- Responsive Navigation Bar
- Futuristic Hero Section
- Typing Animation
- Glassmorphism Design
- Feature Cards
- About Section
- Statistics Section
- Services Section
- Contact Form
- Footer with Social Media Icons
- Responsive Design
- Smooth Animations

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 🚀 How to Run

1. Download or clone the repository.
2. Open the project folder in Visual Studio Code.
3. Open `index.html`.
4. Click **Go Live**.
